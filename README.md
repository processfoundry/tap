# Processfoundry Tap

## How do I install these formulae?

`brew install processfoundry/tap/<formula>`

Or `brew tap processfoundry/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "processfoundry/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
